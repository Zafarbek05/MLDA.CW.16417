# 6COSC017C-n Machine Learning and Data Analytics Coursework

### Student ID: 00016417

## Overview

The aim of this coursework is to design and deliver an end-to-end data analysis and machine learning solution, including a written report, a reproducible Jupyter workflow, and a deployed Streamlit application.

This repository contains all the necessary files and source code for the project.

## Project Structure

This project follows a standard data science project layout. Key directories include:

*   `notebooks/`: For initial exploration and prototyping.
*   `src/`: For reusable source code modules.
*   `data/`: For raw and processed datasets.
*   `models/`: For trained machine learning models.
